## Development

Tools:

* Node.js: v11
* Yarn: v1.12


Start:

    yarn install
    
    
Building:

    yarn build
    
    
Before commit:
    
    yarn commit-check
    
    
Publishing:

    yarn commit-check
    yarn clean && yarn build
    yarn publish-check
    npm run publish-do --otp=<CODE>
